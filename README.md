# Research Paper Summaries


Structured summaries of various research papers revealing each paper's question, methodology, key findings and its significance


# Format
Question: what was the paper trying to find out
Method: Methodology used 
Key Result: The main finding of the paper
Why It Mattered: What difference did the paper create
One Critique: Wonderous or unconvincing fact
